SYSTEM LABEL

AFFECTOS

Behavioral Feed System

Medium: Interactive Web Interface

Date: 2026

AffectOS is a speculative interface that reveals how social media platforms shape user emotions and behavior.

When a user clicks "like," "comment," or "share," the system interprets this as interest. It then adjusts the feed to show more similar content. Over time, this creates a feedback loop: repeated exposure reinforces emotional states such as anxiety, productivity pressure, or social comparison.

The system does not respond to what users need. It amplifies what they react to.

Are we choosing what we see,
or is the system choosing for us?
You clicked. The system learned.
